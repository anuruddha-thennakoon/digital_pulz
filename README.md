# digital_pulz
This is an angular based application for pharmacies.That meets the general requirements of pharmacies in sri lanka.

# Run
```
mongod

npm install
npm start
```
Runs an express server on port 3000 (http://localhost:3000/)

# Services
```
http://localhost:3000/users

  {
	  "firstName":"John",
	  "lastName":"Doe",
	  "date":"2017-05-31"
  }
  
```
